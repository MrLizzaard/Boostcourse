# 설명
이 사이트는 페이지를 라우팅 할 때의 애니메이션을 사용하여 자연스러운 페이지 전환을 연습한 사이트이다.

애니메이션을 사용할 때에는 gsap과 dogstudio/highway라는 이름의 라이브러리를 사용.
